# Security Policy

## Supported Versions

At present, this project is in active development and no official release versions have been published.
Security updates and patches are applied continuously to the main development branch.

| Version | Supported          |
| ------- | ------------------ |
| Unreleased (development branch)   | :white_check_mark: |
| All released versions (N/A)  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

How to report:
Submit a report via GitHub Issues
 using the “Security” label.

Response time:
We aim to acknowledge valid reports within 2 working days and provide updates on the status of the fix within 7 working days.

What to expect:

Valid vulnerabilities will be investigated promptly.

Once verified and fixed, details of the vulnerability and resolution will be documented in the project’s changelog or release notes.

Please do not disclose the vulnerability publicly until it has been addressed.
